# pangenome

HPRC pangenome haplotypes. Originally split from the jungle store; also built independently via `data_loaders/riva_pangenome_analysis/`.

## Sources

Extracted from `repos/refget/data_loaders/pangenome_digests.json`, which mirrors the PEPhub project `nsheff/pangenome_fasta`. FASTA URLs point to Ensembl rapid-release unmasked genome assemblies. 96 haplotypes from the HPRC Year 1 draft assemblies (2023).
