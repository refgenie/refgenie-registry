# vgp

VGP (Vertebrate Genomes Project) vertebrate genomes. Originally split from the jungle store.

## Sources

531 vertebrate genome assemblies from the Vertebrate Genomes Project and affiliated efforts (Bat1K, Wellcome Sanger Institute, etc.). FASTA files are hosted on the NCBI FTP server. Source metadata was extracted from FHR (FAIR Header Reference) records attached to the deployed VGP refgetstore on S3.

Note: The deployed VGP store contains 611 collections total; 80 lack FHR metadata and could not be traced back to a source FASTA. These are likely duplicates or alternate haplotypes that were loaded without provenance tracking.
